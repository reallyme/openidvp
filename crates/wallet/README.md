# reallyme-openid4vp-wallet

Wallet-side OpenID4VP request parsing, Request Object verification boundaries,
transport validation, and presentation derivation hooks.

Network access, trust resolution, JOSE verification, and ZK proving are injected
by hosts or higher-level SDK crates.
