# reallyme-openid4vp-formats

Format-specific OpenID4VP presentation glue for mdoc and ZK presentation entries.

Concrete cryptography, envelope parsing, and ZK backends stay injected or
delegated; this crate checks protocol binding and fail-closed format behavior.
